Beerminology

In the Beerminology tab, I'll explain some basic terms you may need to know as you browse through the following content.

Czech-in Map

Czech-in Map is the main application of the website. User can visualize all venues I've visited and all breweries I've tasted beer from. The left panel provides a variety of filters that can be used to visualize only certain data sets. The right panel then displays check-in details after clicking on a specific beer name in the feature's pop-up windows.

Cartograms

Cartograms tab provides 3 drop-down menu options showing cartographic maps for specific areas. 2 phenomena are displayed, total number of check-ins and total number of different breweries for area units.

Statistics

In the last tab you will find several check-in statistics described in tables and graphs. More statistics will be added over time.

This website was created as part of a semester project on WebGIS Applications in SP during my Erasmus stay at the University of Porto.
